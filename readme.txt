== Description ==

A sidebar menu allows you to organize information vertically, it is a part of a web page that makes it easy to place navigation or display links to help customers find important information easily. Also, it improves the availability area of the site. When the content on the left side of the page is scrolled, the right sidebar navigation node is activated in association; when the side navigation menu is clicked, the corresponding content on the left side will also be scrolled to the top of the visible area.

== Installation ==

Unzip plugin files and upload them under your '/wp-content/plugins/' directory.

Resulted names will be: './wp-content/plugins/sidebar-menu/*'

Activate plugin at "Plugins" administration page.
